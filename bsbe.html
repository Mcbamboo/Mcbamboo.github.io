Changelog

new version 2.8
in this version there are many updates and changes to all assets but not performance, so there will be more updates to improve performance.

  • new sky reflection on water
  • new cloud reflection on the water
  • new water bump/wave (maybe heavier)
  • increase the wet effect when it rains
  • new fog
  • now the color of the fog follows the gradation color of the sky
  • new sky scattering
  • new tonapping method
  • more balanced lighting and color
  • new cloud (using voronise maybe heavier)
  • improve sun reflection on water
  • new effect of sky light reflection on dark areas outdoors
  • now the moon has a phase that change every night changes
