Mirai Shaders BE

           

Mirai shader is a shader (resource pack) for the Minecraft BE with focuses on the quality of the environment (not really correct) and have some features such as volumetric cloud, sky, waterbump, caustic etc..

This shader is not related to minecraft default time because all the lighting elements in this shader use atmospheric scattering, so an accurate value of the sun's position is needed. Here I tricked it by creating a fake position (value) that can be set by the user.


Shader Settings

to make some changes in the shader go to root directori of this shader > shaders > glsl > common.glsl
You can also change the time cycle or position of sun
