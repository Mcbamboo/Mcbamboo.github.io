Mirai Shaders BE

           

Mirai shader is a resource pack for the Minecraft bedrock edition. This shader focuses on the quality of the environment, especially the sky, besides that this soldier is very heavy, so i recommend that users run this shader on high-end devices.
This shader is not related to minecraft default time because all the lighting elements in this shader use atmospheric scattering and because of that, an accurate value of the sun's position is needed. Here I tricked it by creating a fake position (value) that can be set by the user.


Shader Settings

to make some changes in the shader go to root directori of this shader > shaders > glsl > common.glsl
