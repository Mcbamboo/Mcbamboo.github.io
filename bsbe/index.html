Bicubic Shaders BE

A shaders (resource pack) for Minecraft: Bedrock Edition that aims to be beautiful look with colorful tone and some visual features, this pack also known as BSBE

Main Features:

  • Semi volumetric fluffy clouds
  • Sky gradient
  • Sky based border fog
  • Vibrance color tone
  • Dynamic diffuse light and ambient color change (sunrise, sunset, night, rain)
  • Water wave (normal map)
  • Sky and clouds reflection on water
  • Lightmap based shadow
  • Underwater caustic

Latest Version: 3.4

Installation

  • For MCBE Windows with Render Dragon Engine:

    1. Download BetterRenderDragon here, that tool can add some features to the Render Dragon engine, especially the feature to load material bin files through resource pack, see more
    2. Unzip it to the folder you want, then double-click the 'LauchMinecraft.bat' file
    3. After the game is opened, minimize it
    4. Install the shaders with double click on 'Bicubic-Shaders-BE-3.4.mcpack'

  • For MCBE Android with Render Dragon Engine:

    1. Download patched apk here
    2. Install the apk
    3. Install the shaders (resource pack) as usual

  • For MCBE Android with HAL Engine (before Render Dragon):

    1. Just install the shaders (resource pack) as usual

FAQs

Can this pack run on MCBE with HAL engine?
Of course, there are BSBE versions here that support the HAL engine and Render Dragon: BSBE 3.0, 3.2, and 3.3
For iOS?
You can use the old version of MCBE which still uses the HAL engine (MCBE 1.18.12)
Can I use this pack on MCBE Preview?
Of course, but maybe you will get many bugs and all BSBE versions here have only been tested on the release version
